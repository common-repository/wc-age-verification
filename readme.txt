This plugin allows you to set age restrictions on orders

== Description ==

This plugins a calendar on the woocommerce payment page.
The minimum age is set in the administrator section within the wocoomerce settings page.
The plugin validates the age and if it is less than the age required, it will not allow the customer to check out.

== Installation ==

= Installation Instructions =

1. Go to Plugins > Add New > Search
2. Type Registration form eu in the search box and hit Enter
3. Click on the button Install and then activate the plugin

= Manual Installation =

The manual installation method involves downloading our plugin and uploading it to your webserver via your favourite FTP application.

The WordPress codex contains [instructions on how to do this here]:
http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation
